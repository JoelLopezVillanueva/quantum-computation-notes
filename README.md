# Quantum Computing and Quantum Information

This repository serves as my workspace, theoretical notes, and practical development notebooks in quantum computing. The project is under active development and will be continuously updated throughout the academic year.

## Repository Structure & Content

### 1. Theoretical Notes (LaTeX)
Typeset notes in LaTeX based on the following academic curricula:
* **Undergraduate Quantum Computation** (Carnegie Mellon University) – Prof. Ryan O'Donnell. Focused on theoretical computer science, computational complexity, and the mathematical analysis of algorithms.

### 2. Lab Notebooks (Jupyter Notebooks)
Practical implementations in Python and development notes based on scientific literature in quantum optimization and Quantum Machine Learning (QML):
* **Quantum Optimization (QUBO/Ising):** Mathematical formulation and solving of NP-hard combinatorial problems (Max-Cut, Knapsack, Graph Coloring, TSP) via the Ising model. Benchmarking and testing using local quantum annealing simulators (`dwave-neal`).
* **Quantum Machine Learning (QML):** Transitioning from classical neural network architectures (TensorFlow/Keras) toward hybrid models and supervised quantum learning (CQ).

## Repository Organization

* `/LaTeX/` - Source `.tex` files of the academic notes.
* `/PDFs/` - Compiled final documents ready for reading.
* `/Notebooks/` - Jupyter Notebooks containing the optimization and QML simulations.
* `/TikZ/` - Vector source code for quantum circuit diagrams and schematics.
